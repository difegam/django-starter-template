{# Button Component A reusable button component with support for daisyUI variants, sizes, and HTMX attributes. Usage: Click me Large Primary Small Secondary Delete ⋮ Props: variant: Button style (primary|secondary|accent|neutral|ghost|link|outline) - default: 'primary' size: Button size (xs|sm|md|lg|xl) - default: 'md' type: Button type attribute (button|submit|reset) - default: 'button' disabled: Whether button is disabled - default: false wide: Make button wider with padding - default: false block: Make button full-width - default: false square: Make button square shape - default: false circle: Make button circular shape - default: false All HTML attributes (class, id, hx-*, data-*, aria-*, etc.) are forwarded via {{ attrs }}. Additional CSS classes can be added via the class attribute and will be merged. #}
